# comfort analysis

Codes developed to make a comfort analysis based on telemetry and bus passager perception.

The order in which the notebooks must be executed is as follows:

1 - descritive

2 - preprocessing

3 - consensus

4 - annotation

5 - train trees

6 - markov

7 - ensemble
